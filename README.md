# Secret Verification Platform

Mobile-first real-time secret sharing app.

## How to Deploy on Railway (from mobile)

1. Push this repo to GitHub
2. Go to railway.app → New Project → Deploy from GitHub
3. Connect your repo
4. Add these Environment Variables:
   - GMAIL_USER
   - GMAIL_APP_PASSWORD (create app password in Google account → 2FA → App passwords)
5. Deploy
6. Open the live URL → use /api/create-session to start a session

Test as holder and customer on the same link!
